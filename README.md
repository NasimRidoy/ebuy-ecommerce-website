# eBuy - eCommerce Platform

**eBuy** is a full-featured eCommerce platform built with the **MERN stack** (MongoDB, Express.js, React.js, Node.js) and **Redux** for state management. It allows users to browse products, manage their cart, place orders, and make payments online. Admins can manage products, users, and orders efficiently through an intuitive dashboard.

---

## Features

- Full-featured shopping cart  
- Product reviews and ratings  
- Top products carousel  
- Product pagination  
- Product search feature  
- User profile with order history  
- Admin product management  
- Admin user management  
- Admin order details page  
- Mark orders as delivered option  
- Checkout process (shipping, payment method, etc.)  
- PayPal / credit card integration  
- Database seeder (products & users)  

---

## Tech Stack

**Frontend:**  
- React.js  
- Redux for state management  
- Bootstrap / CSS for styling  

**Backend:**  
- Node.js  
- Express.js  

**Database:**  
- MongoDB  

**Other:**  
- PayPal API integration for payments  
- JWT authentication
